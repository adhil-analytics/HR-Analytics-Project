# HR Analytics Dashboard Project

## Project Overview
This project analyzes HR employee data to identify attrition trends, department distribution, salary analysis, and overtime impact.

## Tools Used
- Excel
- SQL
- Tableau

## Project Steps
1. Cleaned HR dataset using Excel
2. Performed SQL analysis and generated insights
3. Built interactive Tableau dashboard

## Key Insights
- R&D department has highest employee count
- Overtime employees show higher attrition
- Majority employees remain in company
